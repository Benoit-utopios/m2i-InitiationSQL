### Exercice récapitulatif – 20 questions

#### **Section 1 : Sélection et colonnes**
1. Écrivez une requête pour récupérer les colonnes `first_name` et `last_name` de la table `Users`.
2. Comment sélectionner toutes les colonnes d'une table `Products` ? Écrivez la requête correspondante.

#### **Section 2 : Clause WHERE et opérateurs de comparaison**
3. Écrivez une requête pour sélectionner les utilisateurs dont le prénom est `Alice` dans la table `Users`.
4. Récupérez toutes les personnes de la table `Employees` ayant un âge supérieur ou égal à 30.
5. Quelle requête permet de trouver tous les produits de la table `Products` dont le prix est différent de 100 ?

#### **Section 3 : Comparaison de texte**
6. Écrivez une requête pour sélectionner les utilisateurs dont le prénom est `Bob` et dont le nom de famille est `Smith` dans la table `Users`.

#### **Section 4 : Projections et sélections**
7. Dans la table `Books`, sélectionnez uniquement les colonnes `title` et `author` pour les livres publiés après l'année 2000.

#### **Section 5 : Opérateurs logiques**
8. Écrivez une requête pour récupérer les utilisateurs nés à `Paris` ou `London` dans la table `Users`.
9. Dans la table `Users`, récupérez les utilisateurs ayant un âge supérieur à 25 **et** un salaire inférieur à 4000.
10. Écrivez une requête qui retourne tous les utilisateurs ayant un prénom qui n'est pas `Alice`.

#### **Section 6 : Combinaisons d'opérateurs**
11. Dans la table `Users`, trouvez tous les utilisateurs nés à `Paris` **et** ayant un salaire compris entre 2000 et 5000, mais **pas** avec le métier de `Engineer`.
12. Écrivez une requête qui retourne les utilisateurs avec un salaire supérieur à 4000, mais pas ceux qui ont comme job `Doctor` ou `Lawyer`.

#### **Section 7 : DISTINCT et valeurs uniques**
13. Écrivez une requête pour récupérer toutes les professions uniques dans la table `Jobs`.
14. Dans la table `Users`, récupérez toutes les combinaisons uniques de prénoms et noms.

#### **Section 8 : IN et BETWEEN**
15. Récupérez tous les utilisateurs de la table `Users` dont le prénom est soit `Alice`, soit `Bob`, soit `Charlie`.
16. Écrivez une requête pour trouver tous les utilisateurs dont l'âge est compris entre 18 et 25 ans.
17. Dans la table `Users`, trouvez tous les utilisateurs n'étant **pas** âgés entre 30 et 40 ans.

#### **Section 9 : Tri des résultats**
18. Écrivez une requête pour récupérer tous les utilisateurs triés par âge de manière décroissante.
19. Récupérez les utilisateurs triés par `last_name` (ordre alphabétique) puis par `age` (ordre croissant).

#### **Section 10 : Limiter les résultats**
20. Dans la table `Products`, écrivez une requête pour récupérer les 5 produits les moins chers.
